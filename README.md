# HireReady

Placement readiness analytics — skill gap detection, employability 
scoring,
and ML insights for students entering the job market.

Built as part of my Foundations of Data Analytics course.

## How to run

pip install -r requirements.txt
streamlit run app.py

## Features
- Student profile with employability score
- Skill gap detection against company requirements
- Data wrangling lab (missing values, outliers, PCA)
- EDA with correlation heatmaps and distributions
- ML: KMeans clustering + logistic regression predictor
- Stats: t-test, chi-square, ANOVA, Pearson/Spearman

## Tech
Python, Streamlit, Pandas, NumPy, scikit-learn, SciPy, Seaborn
